Media API Hub

A collection of media download and search APIs

Music Search
GET /search

Search for music tracks with metadata from YouTube.

q or song
string (required)
Search query (e.g., "Shape of You" or "Ed Sheeran - Shape of You")
Example Request
GET /search?q=Shape+of+You
GET /search?song=Ed+Sheeran+-+Shape+of+You
Try it out
YouTube Video Downloader
GET /youtube

Download YouTube videos in highest quality.

url
string (required)
Valid YouTube video URL (e.g., https://www.youtube.com/watch?v=dQw4w9WgXcQ)
Example Request
GET /youtube?url=https://www.youtube.com/watch?v=dQw4w9WgXcQ
Try it out
Note: This will directly download the video file.
TikTok Downloader
GET /tiktok

Download TikTok videos without watermark.

url
string (required)
Valid TikTok video URL (e.g., https://www.tiktok.com/@user/video/123456789)
Example Request
GET /tiktok?url=https://www.tiktok.com/@user/video/123456789
Try it out
Note: This will directly download the video file.
